## Features

- Light/dark mode toggle
- Invoicing
- Purchasing
- Dashboard
- Live previews
- Fullscreen mode
- Cross platform

## Tech Stack

**Client:** React, NextJS, ShadcnUI, TailwindCSS, MagicUI

**Server:** Node, tRPC

## Run Locally

Clone the project

```bash
  git clone https://github.com/mhmadamrii/serenity.git
```

Go to the project directory

```bash
  cd serenity
```

Install dependencies

```bash
  npm install
```

Generate Database

```bash
  npm run db:push
```

Start the server

```bash
  npm run dev
```
